Binary Search

Overview Visualization Implementation Application

You may be wondering, "when will I ever use binary search?"
Well, we actually use binary search in our every day lives as well as in programming!

One real world example is a simple game where you guess the number someone is thinking of.
Like the name suggests, one person thinks of another while another person tries to guess what number it is.
If the number guessed is less than the target, we instantly start guessing numbers larger than our initial guess.
This pattern goes on until the target number is found.

Another real world example is searching for a book in a library.
Since the books are categorized in some way, we can search for books using the same method.
